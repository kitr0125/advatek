Real-Time Antimalware
Always On
Every file that opens, moves, or runs on your workstation is scanned in real time before it can cause harm. This includes files from USB drives, email attachments that were saved to disk, downloads, and files accessed from shared network drives.
  • Scans all file types — not just known risky extensions
  • Checks inside compressed and archived files (ZIP, RAR, etc.)
  • Detects threats hiding in process memory — not just on disk
  • Identifies keyloggers and potentially unwanted software
  • Detected files are quarantined — reviewed by Advatek before any permanent action
🔒
Ransomware Protection & Recovery
Auto-Recovery Enabled
Ransomware encrypts your files and demands payment. GravityZone monitors for abnormal encryption activity on your workstation and across your network drives. If ransomware is detected, it's stopped and your files are automatically restored from protected backup copies — without you needing to do anything or Advatek needing to manually intervene.
  • Monitors local file activity for encryption patterns
  • Also monitors network drives — catches ransomware spreading from another infected machine
  • Blocks Windows encryption system abuse (a common ransomware technique)
  • Automatic file recovery — no ransom payment, no manual restore process
🌐
Web & Phishing Protection
Block Mode Active
Phishing is the most common way attackers gain access to business accounts and networks — your staff receives a convincing email, clicks a link, and lands on a fake login page. GravityZone intercepts and blocks these sites before the page even loads, so the threat is eliminated before any interaction can happen.
  • Blocks phishing and fraud pages — not just reported-only, but actively stopped
  • Scans all web traffic for malware before it reaches the browser
  • Evaluates HTTPS sites without requiring full SSL interception
  • Users see a blocked page message — no silent failure
📡
Network Attack Defense
All Attack Categories Blocked
Once an attacker gets into one machine on a network, they typically try to spread to others using known network exploitation techniques. GravityZone monitors network traffic at the workstation level and blocks these patterns — limiting the blast radius if any machine in the environment is compromised.
  • Blocks network-based exploitation attempts (including EternalBlue/SMB exploits)
  • Detects and stops brute force credential attacks over the network
  • Blocks lateral movement techniques — prevents one infected machine from compromising others
  • Stops command-and-control communication patterns used by malware
⚙️
Behavioral Threat Detection
Advanced Threat Control
Traditional antivirus relies on known threat signatures — it catches what it recognizes. Behavioral detection watches what programs actually do, not just what they look like. This catches new, unknown, and fileless threats that never touch disk and wouldn't be recognized by a signature scan.
  • Monitors running processes for suspicious behavior patterns
  • Blocks attempts to steal Windows credentials from protected system areas
  • Catches malware executing through PowerShell, scripts, and command-line tools
  • Detects threats operating at the Windows kernel level
  • Catches fileless attacks — malware that runs entirely in memory without creating a file
🔬
Agent Self-Protection
Anti-Tampering Active
One of the first things a sophisticated attacker tries to do after gaining access to a machine is disable the security software. GravityZone protects itself against this. If something attempts to shut down the security agent, the workstation is immediately isolated from the network — cutting off the attacker's access while preserving your other machines.
  • Blocks known techniques used to kill or disable endpoint security agents
  • If tampering is detected: workstation is network-isolated immediately
  • Isolation prevents the attacker from moving laterally or communicating with external servers
  • Advatek is alerted immediately when any tampering event occurs
Scheduled Scans
⚡
Daily Quick Scan — 12:00 PM
Every day at noon, GravityZone runs a lightweight scan that checks for active malware running in system memory. It typically completes in under a minute and has no noticeable impact on your workstation's performance. It catches threats that slipped through in real-time and are now active in memory.
🔍
Weekly Full Scan — Sunday 2:00 AM
Once a week on Sunday morning, GravityZone runs a full scan of every drive, every file, boot sectors, system memory, the registry, and archived files. It runs quietly in the background at the lowest possible CPU priority. Workstations should be left powered on overnight for this to complete. If a machine was off, the scan runs the next time it comes online.
How Advatek Monitors Your Endpoint Security
📋
Weekly Health Verification
Every Monday morning, Advatek's automated systems check that the GravityZone agent is running, registered, and using current threat definitions on every managed workstation. If anything is wrong — service stopped, definitions outdated, agent unregistered — a support ticket is created automatically and Advatek investigates.
🚨
Immediate Alerts on Critical Events
Advatek is alerted in real time when GravityZone detects a ransomware attack, a malware outbreak across multiple machines, or an attempt to disable the security agent. These events are treated as active incidents — not routine tickets — and response begins immediately.
🔄
Automatic Agent Recovery
If the GravityZone security service stops unexpectedly — due to a system error, update conflict, or any other reason — Advatek's remediation script automatically attempts to restart it. If the restart fails, a ticket is created and Advatek investigates manually. You stay protected without needing to call us.
🔐
Uninstall Protection
The GravityZone agent cannot be removed by a regular user or even a standard administrator without Advatek's credentials. This prevents users from accidentally or intentionally removing endpoint protection — and ensures that an attacker who gains admin access cannot simply uninstall the security software.
What Happens When a Threat Is Detected
Malware or Suspicious File
The file is quarantined immediately — moved to a secure, isolated area where it cannot run or spread. Your work is not interrupted. Advatek reviews quarantined items and determines whether the file is a genuine threat or a false positive. If it's a real threat, it's deleted. If it's a false positive (legitimate software that was incorrectly flagged), it's released and an exclusion is added.
Ransomware Detected
The ransomware process is stopped immediately. GravityZone automatically restores any files that were affected from protected backup copies — no ransom payment, no data loss, no manual recovery process. Advatek receives an immediate alert and investigates how the ransomware entered the environment to prevent recurrence.
Phishing or Malicious Website
The connection is blocked before the page loads. The user sees a blocked page notification. No credentials are entered, no malicious content is downloaded. No further action is needed from the user — the threat was stopped at the network level before it had any opportunity to cause harm.
Agent Tampering Attempt
If something attempts to disable the GravityZone agent, the workstation is immediately isolated from the network — cutting off both attacker access and malware communication. Advatek is alerted instantly and treats this as a high-priority security incident. The machine is investigated before network access is restored.
What's Not in SMB Standard

The SMB Standard endpoint security configuration is designed for general business environments. Two capabilities are available but not included by default:

HyperDetect (Advanced Machine Learning): An additional layer of pre-execution threat blocking using aggressive machine learning. Effective, but carries a higher false-positive risk on mixed business software environments. Available as an upgrade — ask Advatek if your environment warrants it.

USB / Device Control: Blocking USB drives, Bluetooth devices, and external storage is available but not applied by default, as most businesses have legitimate uses for these devices. If your business handles sensitive data and you want to restrict what can be plugged in, Advatek can enable this on a per-device or per-site basis.

Healthcare environments are handled under a separate, more stringent hardening suite that includes device blocking, HyperDetect, HIPAA-aligned audit logging, and additional safeguards for clinical applications. If your environment is subject to HIPAA, contact Advatek to discuss the Healthcare hardening program.
