Security Hardening Stack Deployed & Documented
GravityZone Healthcare Policy
  • Device Control USB drives, Bluetooth, portable media, and optical drives blocked on all clinical workstations
  • Network Attack Defense All 5 ATT&CK categories set to Block — Initial Access, Credential Access, Lateral Movement, and more
  • Advanced Anti-Exploit eClinicalWorks, Citrix Workspace, and WebView2 individually protected — kill-process response on exploitation
  • Hyper Detect Ransomware and exploit detection at Aggressive level with automatic block response
  • Antiphishing & Web Scan Both set to Block (not report only) — malicious sites stopped before the user can interact
  • Nightly Full Scan 2:00 AM — boot sectors, process memory, registry, rootkits, archives
All module settings are documented in the GravityZone Healthcare Hardened Policy reference, available on request.
Syncro RMM Hardening Suite
  • BitLocker Encryption Enabled at provisioning with TPM + recovery password; key escrowed to our RMM immediately
  • Windows Security Hardening SMBv1 off, TLS 1.0/1.1 disabled, NTLMv2 enforced, WDigest credential caching disabled
  • HIPAA Audit Policy 22 audit subcategories enabled — logon, object access, removable storage, privilege use, and more
  • Automatic Workstation Lock Machine-level 10-minute lock policy — overrides user settings, cannot be bypassed
  • Weekly Compliance Checks 6 scripts run every Monday — BitLocker, patch status, admin accounts, GZ health, failed logins, disk/uptime
  • Auto-Ticket on Findings Any compliance deviation automatically creates a support ticket — no manual review required
All 17 scripts are documented in the Syncro RMM Healthcare Hardening reference, available on request.
